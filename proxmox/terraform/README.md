# Instrucciones

1. Crear un token en proxmox y asegurarse que la casilla de **Privilege Separetion** este **destildada**

2. Agregar el api key y el secret generado en el archivo **.env**

3. Ejecutar

    ```bash
    . .env
    ```

4. Ejecutar

    ```bash
    make i
    make p
    make a
    ```
